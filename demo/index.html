AnetsTerminal

An embeddable web-based terminal emulator

anets-terminal
Embeddable terminal emulator — zero dependencies, Canvas-based rendering, full ANSI color support (16-color, 256-color, true color), scrollback, mouse selection, cursor styles, and a complete xterm.js-compatible API.

Quick Start

// Import
const term = new AnetsTerminal({ cols: 80, rows: 24 });

// Mount to DOM
term.open(document.getElementById('terminal'));

// Write output
term.write('\x1b[32mHello, World!\x1b[0m\r\n');

// Handle user input
term.on(TerminalEvent.Data, data => {
  // Send data to your backend/PTY
  websocket.send(data);
});

// Receive output from backend
websocket.onmessage = (e) => {
  term.write(e.data);
};
