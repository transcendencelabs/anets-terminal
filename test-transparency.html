🔍 Transparency Test

If transparency works, you should see the colorful gradient background through the terminal text.

transparency-test

What to look for:

  • ✅ Text should be readable with the gradient visible behind it
  • ✅ Cells with explicit background colors (like \x1b[41m) should still show their color
  • ✅ Default cells should be transparent (no black box behind each character)
